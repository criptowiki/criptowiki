Política de privacidad

Tu privacidad nos importa.

El sitio web https://criptowiki.com no registra ningún tipo de información personal sobre sus usuarios. Puedes ver el código fuente del mismo en GitHub para ver que no hay código incrustado de servicios externos como Google Analytics, Google Fonts, ni tampoco APIs de redes sociales como Facebook y Twitter.

Sin embargo, al estar alojado en GitHub Pages, es posible que la empresa detrás del servicio, GitHub Inc, sí registre información personal, incluyendo direcciones IP.

Última actualización: 9 de abril 2022, reflejando cambio de dominio. Cualquier cambio en la política de privacidad se notificará en https://criptowiki.com/politica-de-privacidad/
